\# 🛣️ Toll Pass Token System



A professional toll plaza management system with QR code integration, built with Streamlit.



\## ✨ Features



\- ✅ User registration with 8-digit ID

\- 💰 Purchase toll crossings (10, 20, 50, 100)

\- 📱 QR code generation for each user

\- 🔍 Real-time QR code scanning simulation

\- 💳 Balance tracking and management

\- 📊 System analytics dashboard

\- 🤖 License plate detection (AI-powered)



\## 🚀 Tech Stack



\- \*\*Python 3.8+\*\* - Core language

\- \*\*Streamlit\*\* - Web framework

\- \*\*OpenCV\*\* - Computer vision

\- \*\*SQLite3\*\* - Database

\- \*\*Pandas/NumPy\*\* - Data analytics

\- \*\*Razorpay\*\* - Payment gateway
